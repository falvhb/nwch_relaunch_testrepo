    Tracker

    Does the tracking for all applications (currently Net Metrix, Google Analytics, Chartbeat).

    For tracking Net Metrix make sure the host under which the page is requested is configured as live host in ./app/node_modules/tracking/config.json. If the host is not whitelisted tracking is logged to console.

    public static trackPageView(data: object) source

    Tracks page views.

    Net Metrix: Tracks Net Metrix or logs tracking to console for non-live hosts.

    Google Analytics: ...

    Params:

    NameTypeAttributeDescription
    data object

    Configuration data.
    data.domain string

    The identifier for the Audit. For "Nordwestschweiz Netz" this is 'aznetz'.
    data.path object

    The path information object.
    data.path.product string

    The product. Either 'live', 'epaper' or 'service'.
    data.path.sitename string

    The newsportal. One of: 'aaz', 'bat', 'bzb', 'blz', 'gtb', 'liz', 'ot', 'soz'.
    data.path.path string

    The path of the url of the current page.
    data.path.view string

    The view. For pageview tracking, e.g. path.event=='pageview': page or 'page-noscript' (when JavaScript disabled). For event tracking, e.g. path.event!='pageview': '[name of the component that receives the user event]'.
    data.path.event string

    The user generated event. 'pageview' for page view tracking or '[name of event]'.
    data.path.id string
    • optional

    The ID of the component or the page.
    data.path.index string
    • optional

    The index of the request. Example: For the image gallery this is the index of the image requested.
    data.referer string
    • optional

    The referer URL.
    data.screenWidth integer
    • optional

    The width of the devices' screen in pixels.
    data.screenHeight integer
    • optional

    The height of the devices' screen in pixels.

    Example:

    // Track pageview on Aargauer Zeitung.
// Currently only covers Net Metrix tracking.
let data = {
  domain: 'aznetz',
  path: {
    product: 'live',
    sitename: 'aaz',
    path: window.location.pathname,
    view: 'page',
    event: 'pageview'
  }
};

Tracker.trackPageView(data);
