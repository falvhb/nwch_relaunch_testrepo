    breakpoints

    $breakpoints: (
  'small': 450px,
  'medium': 750px,
  'large': 1150px
);

    Description

    Application-wide breakpoints map

    Type

    Map

    Map structure

    Map keyNameMap keyDescriptionMap keyTypeMap keyValue
    key

    Name of the breakpoint

    		String—none
    value

    Breakpoint value expressed in pixels

    		Map—none

    media-expressions

    $media-expressions: (
  'print': 'print',
  'retina': (
    '(-webkit-min-device-pixel-ratio: 2)',
    '(min-resolution: 192dpi)'
  )
);

    Description

    Application-wide static queries map

    Type

    Map

    Map structure

    Map keyNameMap keyDescriptionMap keyTypeMap keyValue
    key

    Name of the query

    		String—none
    value

    Query value

    • key is repeated as value if simple query (e.g. print)
    • list is used for or conditions (e.g. retina)
    		String or List—none
