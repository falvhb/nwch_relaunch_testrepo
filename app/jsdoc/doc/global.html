Methods

value(data)

Tracks page views. Net Metrix: Tracks Net Metrix or logs tracking to `console` for non-live hosts.
Parameters:
Name Type Description
data object Configuration data.
Properties
Name Type Attributes Description
domain string The identifier for the Audit. For "Nordwestschweiz Netz" this is `'aznetz'`.
path object The path information object.
Properties
Name Type Attributes Description
product string The product. Either `'live'`, `'epaper'` or `'service'`.
sitename string The newsportal. One of: `'aaz'`, `'bat'`, `'bzb'`, `'blz'`, `'gtb'`, `'liz'`, `'ot'`, `'soz'`.
view string The view. `page` for `path.event=='pageview'` or `'[component name]'` for event _tracking.
event string The user generated event. `'pageview'` for page view _tracking or `'[name of event]'`.
id string <optional>
 The ID of the component or the page.
index string <optional>
 The index of the request. Example: For the image gallery this is the index of the image requested.
referer string <optional>
 The referer URL.
screenWidth integer <optional>
 The width of the devices' screen in pixels.
screenHeight integer <optional>
 The height of the devices' screen in pixels.

Example
let data = {
  domain: 'aznetz',
  path: {
    product: 'live',
    sitename: 'styleguide',
    view: 'page',
    path: window.location.pathname,
    event: 'pageview'
  }
};

NetMetrixTracker.trackPageView(data);
