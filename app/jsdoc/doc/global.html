Tracker

Does the tracking for all applications (currently Net Metrix, Google Analytics, Chartbeat). For tracking Net Metrix make sure the host under which the page is requested is configured as live host in `./app/node_modules/tracking/config.json`. If the host is not whitelisted tracking is logged to `console`.

value(data)

Tracks page views. Net Metrix: Tracks Net Metrix or logs tracking to `console` for non-live hosts. Google Analytics: ...
Parameters:
Name Type Description
data object Configuration data.
Properties
Name Type Attributes Description
domain string The identifier for the Audit. For "Nordwestschweiz Netz" this is `'aznetz'`.
path object The path information object.
Properties
Name Type Attributes Description
product string The product. Either `'live'`, `'epaper'` or `'service'`.
sitename string The newsportal. One of: `'aaz'`, `'bat'`, `'bzb'`, `'blz'`, `'gtb'`, `'liz'`, `'ot'`, `'soz'`.
path string The path of the url of the current page.
view string The view. For pageview tracking, e.g. `path.event=='pageview'`: `page` or `'page-noscript'` (when JavaScript disabled). For event tracking, e.g. `path.event!='pageview'`: `'[name of the component that receives the user event]'`.
event string The user generated event. `'pageview'` for page view tracking or `'[name of event]'`.
id string <optional>
 The ID of the component or the page.
index string <optional>
 The index of the request. Example: For the image gallery this is the index of the image requested.
referer string <optional>
 The referer URL.
screenWidth integer <optional>
 The width of the devices' screen in pixels.
screenHeight integer <optional>
 The height of the devices' screen in pixels.

Example
// Track pageview on Aargauer Zeitung.
// Currently only covers Net Metrix tracking.
let data = {
  domain: 'aznetz',
  path: {
    product: 'live',
    sitename: 'aaz',
    path: window.location.pathname,
    view: 'page',
    event: 'pageview'
  }
};

Tracker.trackPageView(data);
