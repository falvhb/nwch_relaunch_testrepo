# az-nwch-js

## Setup

```sh
$ gem install scss-lint
$ npm install
```

## Working task

```sh
$ gulp
```

*If you have the LiveReload browser extension, feel free to pass the `--livereload` flag to benefit from instant page reloading on change.*

## Environment build

```sh
$ gulp build
```

## Production build

```sh
$ gulp build --prod
```
